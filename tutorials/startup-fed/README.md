# Startup Federation Template

This directory is a [Cookiecutter](https://cookiecutter.readthedocs.io/) template that generates a minimal DFM federation project so you can start developing adapters and applications without setting up files by hand.

## Quick start

1. From the repository root (or any directory), run:

   ```bash
   cookiecutter tutorials/startup-fed/
   ```

2. Answer the prompts (project name, federation name, app name, etc.). Cookiecutter will create a new project directory with the name you chose.

3. Open the generated project and follow the instructions in its **README.md** to configure DFM, generate code, and run your federation locally.

For full details on directory layout, configuration, and development workflow, see the README inside the generated project.
